\# AIoT 기반 도서관 좌석 사석화 방지 시스템 (Edge-Computing)



이 레포지토리는 도서관 좌석의 사석화 및 분실물 방지를 위한 엣지 서버(Raspberry Pi) 로직과 센서 제어(ESP32) 코드를 포함하고 있습니다.



\## 주요 기능

\- \*\*압력 센서 모니터링\*\*: ESP32를 통해 좌석 착석 여부를 실시간으로 감지 (MQTT 통신).

\- \*\*Edge AI 분석\*\*: YOLOv8n 모델을 사용하여 좌석 내 소지품 유무를 판별.

\- \*\*상태 관리 로직\*\*:

&#x20; - \*\*사석화 의심\*\*: 자리비움 상태에서 짐이 감지될 경우 (15분 경과 시 확정).

&#x20; - \*\*분실물 감지\*\*: 사용자가 앱에서 '퇴실' 버튼을 눌렀으나 소지품이 남아있는 경우 (20분 경과 시 확정).

\- \*\*서버 연동\*\*: FastAPI 서버로 좌석 상태 및 증거 사진(S3 URL) 전송.



\## 파일 구성

\- `main\_system.py`: 라즈베리 파이 통합 제어 및 AI 분석 메인 로직.

\- `esp32\_sensor.ino`: ESP32 압력 데이터 수신 및 MQTT 발행 코드.

\- `requirements.txt`: 시스템 구동에 필요한 파이썬 라이브러리 목록.
